Proceed inside the provided docker image

Building:

1) Create build folder
2) cd build
3) cmake ../
4) make
5) ./GuitarShowcase


Controls:
Press Q and E to rotate guitars  
Press M to make fixed zoom in/out  
While zoomed, W and S scrolls up and down  
Use Back and Next buttons to swap guitar or part  
Press ESCAPE to unselect guitar part  
Press R to reset the view
